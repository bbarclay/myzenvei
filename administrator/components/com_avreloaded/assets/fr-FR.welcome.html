Bienvenue sur AllVideos Reloaded!

Démarrage rapide

Avant d'insérer des fichiers multimédias locaux, utilisez le gestionnaire de média de Joomla (media manager, ou autre gestionnaire de fichiers par FTP) pour créer deux dossiers nommés audio et videos dans votre dossier images/stories. Ensuite, utilisez le gestionnaire de média de Joomla pour télécharger vos fichiers dans ces dossiers. Si cela ne marche pas, il se peut que vous deviez modifier les paramétrages globaux de Joomla pour autoriser le téléchargement de fichiers autres que des images.

Lors de la rédaction de vos articles, utilisez le nouveau bouton Médias AVR au bas de la fenêtre de l'éditeur pour insérer des données multimédias. Pour insérer des vidéos distantes à partir d'un site de mise en ligne de vidéos, collez simplement l'URL de la page affichant la vidéo dans le premier champ de la fenêtre de dialogue venant de s'ouvrir, puis pressez sur RETURN pour démarrer l'auto-complétion. Si l'URL est supporté par AVR, le reste du formulaire sera complété automatiquement. Le dernier champ contient la balise multimédia pouvant être éditée pour ajouter des attributs spécifiques. Finalement, cliquez sur le bouton "Insérer" et le contenu de ce champ sera inséré à la position actuelle du curseur dans l'éditeur.

Pour insérer des fichiers multimédias locaux, utilisez le même bouton Médias AVR, mais sélectionnez le tabulateur Médias Locaux. Le dialogue vous permettra de sélectionner un des fichiers téléchargés précédemment. Après sélection du fichier, les champs restants devraient être remplis automatiquement.

Pour de plus amples informations - spécialement avant d'utiliser les fonctions avancées de l'extension - SVP lisez la documentation inclue. Même si AVR fait le maximum pour faciliter l'inclusion de multimédias dans votre site web, le sujet reste complexe. Pour profiter au mieux de AVR, il est absolument nécessaire de lire la documentation. Celle-ci est accessible en tout temps dans la page d'administration du plugin Content - AllVideos Reloaded ou en cliquant sur le bouton d'Aide dans l'une des pages d'administration de l'extension AllVideos Reloaded.
Le site de démonstration fournit également une collection d'exemples utiles.

Si vous avez besoin d'aide, vous pouvez l'obtenir dans le forum officiel de Joomla, à l'adresse suivante: forum
Si votre fournisseur de vidéos préféré n'est pas supporté par AVR ou si vous avez des idées de développement pour AVR, soumettez une requête dans le suivi du projet.
